Create flash-like effects

// window onload event
MFXWindowLoad load = new MFXWindowLoad();

// append the style to the window onload event
load.addStyle(new MFXStyle("margin-left",-400,0));

// append event to the element that will have the animation
Label lbl = new Label("blah","hello world");
lbl.add(load);

Add smooth scrolling to your web page

// add MFXSmoothScroll to your base page or current page
add(new MFXSmoothScroll());

// now all anchor clicks will have smooth scrolling on your web page

Font Resize Plugin

Dynamically resize your fonts on your page

// webmarkupcontainer is a href link with an image in it.
// increase the font size by +2 for each click, do it only in the dev with id #stage
add(new WebMarkupContainer("larger").add(new MFXFontResizer(2).setContainer("stage")));

// decrease the font size by -2 for each click on the whole page
add(new WebMarkupContainer("smaller").add(new MFXFontResizer(-2)));

Tooltip Plugin

Add dynamic tooltips to your DOM elements

// Plugin automatically inserts the required CSS code needed
Label lbl = new Label("hello","world");
lbl.add(new MFXTips("Hello world!.");

Inline AJAX Validation

onblur input field smoothly turns to a specified color on invalid input

add(new RequiredTextField("blah").add(new MFXValidationHandler("#FFFFFF","#feb4b4")));

// you can also set the wrapper to inline the javascript code on return for the fading
// use this if your form is retrieved via AJAX request 
// since fading function code block won't be written to the base page
add(new RequiredTextField("blah").add(new MFXValidationHandler("#FFFFFF","#feb4b4",true)));
