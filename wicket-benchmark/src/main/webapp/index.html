Wicket versus the Rest

This is a benchmark program to see how Wicket performs against other frameworks. This application was created in 2005 by Phil Kulak, based on the Wicket Phonebook application.

The application shows two screens:

  1. 1. a list of contacts
  2. 2. edit contact information page

In the src/jmeter folder you can find a predefined Apache JMeter test script.

Click on one of the following buttons to test drive each framework using this application.



  • * Wicket 1.2 Session store
  • * Wicket 1.3 Session store
  • * Wicket 1.3 Disk store
